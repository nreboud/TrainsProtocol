This document sums up how to import the TrainsProtocol project from github with EGit installed on Eclipse

1. Select File -> Import...

2. In Git section, choose Projects from Git 
   (Repositories from GitHub seems not to work at the moment)
   Click NEXT

3. Choose GitHub
   Click NEXT

4. Enter TrainsProtocol
   Click Search
   Choose simatic/TrainsProtocol
   Click NEXT

5. Choose the target location

6. If the projects contains a .project and .cproject (and if CDT is installed)
   Choose Import existing projects, choose TrainsProtocol
   Click FINISH

   If it doesn't or if you want to import it as another kind of project
   Click Use the New Project wizard
   Click NEXT
   Choose the kind of project you want
   Click FINISH
